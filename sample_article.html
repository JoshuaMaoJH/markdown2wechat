第一个Python程序

让我们编写一个简单的"Hello World"程序：

# hello.py
def greet(name):
    """问候函数"""
    return f"Hello, {name}!"

if __name__ == "__main__":
    name = input("请输入您的姓名: ")
    message = greet(name)
    print(message)

循环语句

# for循环

    print(f"我喜欢{fruit}")

    print(f"计数: {count}")

函数定义

函数是代码重用的重要方式：

def calculate_area(length, width):
    """
    计算矩形面积

    参数:
        length: 长度
        width: 宽度

    返回:
        面积值
    """
    return length * width

# 使用函数
area = calculate_area(5, 3)
print(f"矩形面积: {area}")

面向对象编程

Python支持面向对象编程：

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        return f"我是{self.name}，今年{self.age}岁"

    def have_birthday(self):
        self.age += 1
        print(f"{self.name}过生日了！现在{self.age}岁")

# 创建对象
person = Person("张三", 25)
print(person.introduce())
person.have_birthday()

异常处理

处理程序运行时可能出现的错误：

try:
    number = int(input("请输入一个数字: "))
    result = 10 / number
    print(f"10除以{number}等于{result}")
except ValueError:
    print("输入的不是有效数字")
except ZeroDivisionError:
    print("不能除以零")
except Exception as e:
    print(f"发生错误: {e}")
finally:
    print("程序执行完毕")
